NAME

lrun_cc - run a (1)/(1)++ compiler protecting include directories

SYNOPSYS

lrun_cc [-m marker] compiler args...

DESCRIPTION

lrun_cc runs the program compiler with any given arguments args... . args are parsed for include or library directories and calls mkdir on these directories and ldepend(1) on marker files within these directories. The goal is to ensure that these directories always exist as this is a requirement of most (1)/(1)++ compiler to try to read include files within them and hence trigger dependencies.
